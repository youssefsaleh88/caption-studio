## Foundation

- [ ] **ربط الحركة بتفضيل النظام وتكوين التوكن:** قراءة `--cap-motion-enable-typing` و`--cap-motion-typing-char-delay` في `Home.jsx` (`TypingTitle`: إما عرض النص كاملًا فورًا أو تأخير متوافق؛ إلغاء خط المؤشّر أو تثبيته). تقليل/إيقاف `animate-pulse` على المؤشّر والخلفيات المتحركة (`bg-mesh-gradient` / `animate-gradient-shift`) تحت `prefers-reduced-motion` (تعطيل عبر class شرطي أو `@media` في `index.css`). _يعدّل: `Home.jsx`, `index.css`; يعيد استخدام: `tokens.css`._

- [ ] **تقارب Tailwind ↔ توكنات اللون:** تدريجياً ربط `theme.extend.colors.dark` و`accent` بقيم `var(--cap-*)` مع دعم شفافية ألفا حيث يلزم، أو توحيد المكوّنات الجديدة على classes تقتصر على `var(...)`. _يعدّل: `tailwind.config.js` و/أو مكوّنات مختارة؛ يعتمد على `tokens.css`._

## Core UI · سلوك منتجي لم يكتمل حرفيًا

- [ ] **خطأ التصدير + إعادة محاولة صريحة:** عند فشل طلب التصدير أو SRT، إبقاء رسالة واضحة (حاليًا موجودة جزئيًا) مع **زر «إعادة المحاولة»** يعيد آخر إجراء (`handleExport` أو تنزيل ذي صلة) بدون مغادرة التبويبة. نفس النمط على تبويب تصدير الموبايل إن لزم المحاذاة. _يعدّل: `ExportBar.jsx`؛ جديد: مفاتيح `ar.json`/`en.json`._

- [ ] **دخول `/editor` بدون جلسة صالحة:** عند `replace` إلى `/`، إظهار رسالة قصيرة على **الصفحة الرئيسية** (banner قابل للإغلاق أو `role=status`) توضّح أن الفتح المباشر للمحرّر غير مدعوم ويُفضّل الرفع من جديد. _(اختياري لكنه يغلق فجوة IA «Flow ب».)_ _يعدّل: `Home.jsx` أو مسار صغير بحالة location من `Editor` — يتطلب تمرير flag بسيط عبر `navigate('/')`._

## Interactions & States

- [ ] **تركيز لوحة المفاتيح وحلقات واضحة:** تطبيق متغيّرات `--cap-focus-ring-*` على تبويبات الموبايل، أزرار ورقة الإعدادات، وأزرار التصدير الأساسية، مع تجنّب تعارض `ring` مزدوج مع Tailwind. _يعدّل: المكوّنات المسمّاة في `Editor.jsx`, `ExportBar.jsx`, جزئياً `StylePanel`._

- [ ] **حالات تحميل إضافية أثناء التصدير:** التأكد أن `busy` يعطّل الأزرار **ويوضّح** «جاري…» بطريقة تُقرأ RTL (بالفعل موجودة جزئياً؛ مراجعة وضوح الهدف 44×44 والنص المرافق على الموبايل). _يعدّل: `ExportBar.jsx`._

## Responsive & Polish

- [ ] **مرور سريع موبايل أولًا (صانع محتوى):** التحقّق أن منطقة اللمس لا تقل عن التوصية، وأن ورقة الإعدادات لا تقصّ حقول sliders؛ تصحيح أي overflow أفقي متبقي. _مراجعة: `Editor.jsx`, `StylePanel.jsx`, قياس Chrome device toolbar._

## خيارات مستقبلية (خارج ضمان v1)
